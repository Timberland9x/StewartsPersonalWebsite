Pocket Translator (Project Slides)

Project Demo Video:
Project Description: Pocket Translator is a mobile application designed for travelers and foreigners to easily translate text from their surroundings. Users can either take a new photo with their camera or upload an existing image from their gallery. The app uses Google Cloud Vision API to detect foreign text in the image and then uses the Google Translate API to provide a translation. The frontend is built with React Native (for cross-platform mobile support), and the backend is powered by Python Flask.
